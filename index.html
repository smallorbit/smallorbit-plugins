the kits

Tools for the whole lifecycle.

Each kit is built for a specific phase. Pick the suite or just what you need.

speckit Plan it — interview-driven planning and issue capture README ↗

Define and capture work as GitHub issues — interview a feature into existence, bulk-convert findings, or quickly file a single issue.

  • /speckit:interview
  • /speckit:spec
  • /speckit:catalog
  • /speckit:issue
swarmkit Execute it — parallel worktree agents resolving issues into stacked PRs README ↗

Resolve GitHub issues with parallel agents — pick what to work on, swarm it in isolated worktrees, merge top-down.

  • /swarmkit:swarm
  • /swarmkit:swarm-plus
  • /swarmkit:next-issue
  • /swarmkit:merge-stack
  • /swarmkit:clean-worktrees
polishkit Polish it — appraise craft, sweep cruft, polish cross-cutting issues README ↗

Codebase quality toolkit — score craft across 5 dimensions, sweep dead code and stale artifacts, polish cross-cutting fixes in one PR.

  • /polishkit:appraise
  • /polishkit:sweep
  • /polishkit:polish
flowkit Git workflow automation — branching, PRs, releases, and hotfixes README ↗

Ship features faster with opinionated Git flow built for Claude Code.

  • /flowkit:create-branch
  • /flowkit:pr
  • /flowkit:merge-pr
  • /flowkit:cut
  • /flowkit:release
sessionkit Throughout — session continuity, planning, and meta-learning README ↗

Hand off context, resume seamlessly, plan work-in-flight to "shipped", and capture reusable patterns from your sessions.

  • /sessionkit:handoff
  • /sessionkit:pickup
  • /sessionkit:roadmap
  • /sessionkit:drive
  • /sessionkit:skillit
squadkit Orchestrate it — multi-agent crews with the roles → squads → crews vocabulary README ↗

Multi-agent team coordination — spawn role-aware crews for execution or read-only discovery, end to end.

  • /squadkit:init
  • /squadkit:spawn-team
  • /squadkit:agent-team-retro
vaultkit Capture it — Obsidian vault skills for notes, projects, and archives README ↗

Read, search, edit, and capture into an Obsidian vault — projects, decisions, and conversation archives, all via the Obsidian CLI.

  • /vaultkit:obsidian
  • /vaultkit:jot
  • /vaultkit:archive-export
  • /vaultkit:project
hookify Guard it — author Claude Code hooks that prevent unwanted behaviors README ↗

Create and manage Claude Code hooks from conversation analysis or explicit instructions — list, configure, and write rules.

  • /hookify:hookify
  • /hookify:list
  • /hookify:configure
  • /hookify:writing-rules
  • /hookify:help
