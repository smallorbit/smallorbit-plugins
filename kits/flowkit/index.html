flowkit

Git workflow automation — branching, PRs, releases, and hotfixes

Ship features faster with opinionated Git flow built for Claude Code.

Read the README ↗ /plugin install flowkit@smallorbit-plugins

What it does

flowkit wraps the full develop → release → main lifecycle into a set of Claude Code skills. It handles branch creation, PR opening with structured bodies, squash-merging, staging cuts, hotfixes, and release tagging — so every merge follows the same convention without manual steps.
