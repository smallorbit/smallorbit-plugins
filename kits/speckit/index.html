speckit

Plan it — interview-driven planning and issue capture

Define and capture work as GitHub issues — interview a feature into existence, bulk-convert findings, or quickly file a single issue.

Read the README ↗ /plugin install speckit@smallorbit-plugins

What it does

speckit is the planning kit. Use /interview to clarify scope, /spec to drive a full requirements interview that files an epic plus child issues, /catalog to bulk-convert review findings into prioritized issues, and /issue to file a single ticket fast — all with duplicate detection and label hygiene built in.

Commands

  • /speckit:interview
  • /speckit:spec
  • /speckit:catalog
  • /speckit:issue
